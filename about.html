Code Cangjie

[ MISSION ]

Code Cangjie is an interactive digital archive exploring the structural logic and visual evolution of Hanzi (Chinese characters). By deconstructing characters into their fundamental radicals, this project aims to bridge ancient etymology with modern interaction design, revealing the underlying philosophical and logical framework of the Chinese writing system.
